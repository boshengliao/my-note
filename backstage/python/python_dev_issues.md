issues:  

1. 生成的一个 `list` 是唯一的, 不能重复使用. 否则, 赋值会被覆盖.  
   * 用 `id(list)` 可查看唯一的id  
   * 被覆盖的情况:  
   
         a = [1, 2, 3]
         b = a
         c = b
         a[0] = 2
         print a
         print b
         print c
         # 输出 a: [2, 2, 3]
         # 输出 b: [2, 2, 3]
         # 输出 c: [2, 2, 3]

         # 类似下面情况也一样会被覆盖
         a = [1, 2, 3]
         b = a * 3
         a[0] = 2
         # 输出 b: [2, 2, 3, 2, 2, 3, 2, 2, 3]
